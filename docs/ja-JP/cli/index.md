# CLI コマンド概要

ZCF の主要 CLI を一覧します。詳細は各ページを参照してください。

- `zcf init`：環境初期化、ワークフロー/テンプレート導入
- `zcf update`：テンプレート/ワークフロー/ツールの更新
- `zcf menu` (`npx zcf`)：インタラクティブメニュー
- `zcf ccr`：Claude Code Router の管理
- `zcf ccu`：利用状況 (ccusage) の表示
- `zcf config-switch` / `zcf cs`：複数設定の切り替え
- `zcf check-updates` / `zcf check`：アップデート確認
- `zcf uninstall`：選択的なアンインストール
