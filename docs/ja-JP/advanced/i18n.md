# 国際化と言語

ZCF は CLI 言語・AI 出力言語・テンプレート言語を個別に設定できる完全な i18n を提供します。用途に応じて自由に組み合わせ可能です。

## 言語レイヤー

1. **CLI 言語**：ZCF 自体の表示言語  
2. **AI 出力言語**：AI 応答の言語  
3. **テンプレート言語**：ワークフロー/プロンプトの言語

互いに独立して設定できます。

## CLI 言語

- サポート：`zh-CN` / `en`
- 優先度：CLI パラメータ > `preferredLang` > 対話選択 > 既定 (`en`)
- 設定例：

```bash
npx zcf --lang zh-CN
npx zcf --all-lang zh-CN   # CLI/AI/テンプレートをまとめて変更
```

## AI 出力言語

- サポート：`zh-CN` / `en` / `custom`
- 設定例：

```bash
npx zcf init --ai-output-lang zh-CN
npx zcf init --ai-output-lang custom   # 例: Reply in Japanese with professional tone
```

`custom` は `CLAUDE.md` / `AGENTS.md` に書き込まれ、出力スタイルにも影響します。

## テンプレート言語

- サポート：`zh-CN` / `en`
- CLI と独立して選択可能。

```bash
npx zcf init --config-lang en           # 英語テンプレートを使用
npx zcf init --lang zh-CN --config-lang en
npx zcf init --all-lang zh-CN           # すべて中国語で統一
```

解決の優先度：`--config-lang` > `templateLang` 設定 > 現在の CLI 言語 > 既定 `en`。

## Codex での言語

Codex でも同じパラメータで設定可能：

```bash
npx zcf init -T codex --all-lang zh-CN
```

## よく使うシナリオ

- **中国語環境**：`npx zcf init --all-lang zh-CN`  
- **中国語 UI + 英語コード**：`npx zcf init --lang zh-CN --ai-output-lang en --config-lang zh-CN`  
- **国際チーム**：`npx zcf init --lang en --config-lang en --ai-output-lang en`
